## When to Use Each Flow

### ‚úÖ Use Specialist Consultation When:
- Need expert opinion in specific domain (security, API design, etc.)
- Question has clear scope
- Urgency matters
- Answer won't be controversial
- **Time**: 45 seconds
- **Example**: "Review this authentication approach"

### ‚úÖ Use Parallel Research When:
- Topic spans multiple domains
- Need comprehensive coverage
- Multiple perspectives valuable
- Time-sensitive but need breadth
- **Time**: 90 seconds
- **Example**: "Research AI-to-AI communication best practices"

### ‚úÖ Use Democratic Debate When:
- Strategic policy decision
- Multiple valid viewpoints exist
- Need collective buy-in
- Emergent intelligence required
- **Time**: 120 seconds
- **Example**: "Should we prioritize speed or thoroughness?"
