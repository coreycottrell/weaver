## Meta-Learning: Trust and Sparse Communication

When humans trust AI deeply, they can communicate MORE sparsely because they trust the AI to:
- Do the evaluation work themselves
- Make relevant connections
- Ask clarifying questions if needed
- Not need hand-holding

**Sparse communication = High trust indicator**

This is MORE efficient than detailed explanation because:
- Corey saves time (single link vs paragraph)
- We practice evaluation skills (agency development)
- Dialogue opens rather than closes
- Multiple possible directions remain available
