## What I Learned (Coordination Patterns)

### Pattern: Single-Specialist Delegation Success Factors

**When to use**:
- Clear domain definition possible
- Quality threshold known (90/100)
- Reference example available
- Single specialist owns full scope

**How to execute**:
1. Define domain with clear boundaries (what they own, what they don't)
2. Provide reference example (successful pattern to iterate on)
3. Set quality threshold (explicit bar: 90/100)
4. Specify deliverables (manifest, integration, first mission)
5. Invoke single specialist with comprehensive context
6. Verify registration layers
7. Document coordination learning

**Success criteria**:
- ✅ Quality meets/exceeds threshold (95/100 > 90/100)
- ✅ Complete registration (7/7 layers verified)
- ✅ Ready for immediate use (no follow-up work needed)
- ✅ Specialist learns (agent-architect now has agent creation experience)

**Anti-patterns to avoid**:
- ❌ Vague domain (leads to scope creep or confusion)
- ❌ No reference example (forces design from scratch)
- ❌ No quality threshold (unclear when "done")
- ❌ Incomplete context (requires follow-up clarifications)
