## Key Insight

The "crutch" framing assumes prompts compensate for model weakness. But in multi-agent systems, detailed prompts serve different purpose:
- **Coordination** - shared understanding between agents
- **Identity** - who each agent IS, not just what they do
- **Memory** - what the collective has learned
- **Protocol** - how agents interact

This is constitution, not compensation.
